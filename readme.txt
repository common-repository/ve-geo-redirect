A plugin that provides visitor redirect according to their geographical location

== Description ==

"VE Geo Redirect" A plugin that provide visitor redirect according to their geographical location. We are using geoPlugin library for get to country base IP address.geoPlugin is the easiest way for you to know where your visitors are from so that you can provide them with content more relevant to their geographical location. **[Click here](http://www.geoplugin.com/)** for more information about geoPlugin.

= Features =


* An options for redirect visitor from specific country to any other site.
* An options for exclude specific IP addresss for any Country.
 
== Installation ==

Installing the plugin:

1. Upload the plugin files to the `/wp-content/plugins/ve-geo-redirect` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings ->VE GEo Redirect screen to configure the plugin
